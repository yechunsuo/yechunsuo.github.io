Portals

Portals 提供了一种很好的将子节点渲染到父组件以外的 DOM 节点的方式。

Portals 这里我个人称他为入口中转器

第一个参数（child）是任何可渲染的 React 子元素，例如一个元素，字符串或碎片。第二个参数（container）则是一个 DOM 元素。
