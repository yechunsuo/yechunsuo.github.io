жЁЎеқ—еҢ–

ES6 жЁЎеқ—дёҚжҳҜеҜ№иұЎпјҢиҖҢжҳҜйҖҡиҝҮexportе‘Ҫд»ӨжҳҫејҸжҢҮе®ҡиҫ“еҮәзҡ„д»Јз ҒпјҢеҶҚйҖҡиҝҮimportе‘Ҫд»Өиҫ“е…ҘгҖӮ

дёҠйқўд»Јз Ғзҡ„е®һиҙЁжҳҜд»ҺfsжЁЎеқ—еҠ иҪҪ 3 дёӘж–№жі•пјҢе…¶д»–ж–№жі•дёҚеҠ иҪҪгҖӮиҝҷз§ҚеҠ иҪҪз§°дёәвҖңзј–иҜ‘ж—¶еҠ иҪҪвҖқжҲ–иҖ…йқҷжҖҒеҠ иҪҪпјҢеҚі ES6 еҸҜд»ҘеңЁзј–иҜ‘ж—¶е°ұе®ҢжҲҗжЁЎеқ—еҠ иҪҪпјҢж•ҲзҺҮиҰҒжҜ” CommonJS жЁЎеқ—зҡ„еҠ иҪҪж–№ејҸй«ҳгҖӮеҪ“з„¶пјҢиҝҷд№ҹеҜјиҮҙдәҶжІЎжі•еј•з”Ё ES6 жЁЎеқ—жң¬иә«пјҢеӣ дёәе®ғдёҚжҳҜеҜ№иұЎгҖӮ

дёҘж јжЁЎејҸ

ES6 зҡ„жЁЎеқ—иҮӘеҠЁйҮҮз”ЁдёҘж јжЁЎејҸпјҢдёҚз®ЎдҪ жңүжІЎжңүеңЁжЁЎеқ—еӨҙйғЁеҠ дёҠ"use strict";гҖӮ

дёҘж јжЁЎејҸдё»иҰҒжңүд»ҘдёӢйҷҗеҲ¶гҖӮ

  • еҸҳйҮҸеҝ…йЎ»еЈ°жҳҺеҗҺеҶҚдҪҝз”Ё
  • еҮҪж•°зҡ„еҸӮж•°дёҚиғҪжңүеҗҢеҗҚеұһжҖ§пјҢеҗҰеҲҷжҠҘй”ҷ
  • дёҚиғҪдҪҝз”ЁwithиҜӯеҸҘ
  • дёҚиғҪеҜ№еҸӘиҜ»еұһжҖ§иөӢеҖјпјҢеҗҰеҲҷжҠҘй”ҷ
  • дёҚиғҪдҪҝз”ЁеүҚзјҖ 0 иЎЁзӨәе…«иҝӣеҲ¶ж•°пјҢеҗҰеҲҷжҠҘй”ҷ
  • дёҚиғҪеҲ йҷӨдёҚеҸҜеҲ йҷӨзҡ„еұһжҖ§пјҢеҗҰеҲҷжҠҘй”ҷ
  • дёҚиғҪеҲ йҷӨеҸҳйҮҸdelete propпјҢдјҡжҠҘй”ҷпјҢеҸӘиғҪеҲ йҷӨеұһжҖ§delete global[prop]
  • evalдёҚдјҡеңЁе®ғзҡ„еӨ–еұӮдҪңз”Ёеҹҹеј•е…ҘеҸҳйҮҸ
  • evalе’ҢargumentsдёҚиғҪиў«йҮҚж–°иөӢеҖј
  • argumentsдёҚдјҡиҮӘеҠЁеҸҚжҳ еҮҪж•°еҸӮж•°зҡ„еҸҳеҢ–
  • дёҚиғҪдҪҝз”Ёarguments.callee
  • дёҚиғҪдҪҝз”Ёarguments.caller
  • зҰҒжӯўthisжҢҮеҗ‘е…ЁеұҖеҜ№иұЎ
  • дёҚиғҪдҪҝз”Ёfn.callerе’Ңfn.argumentsиҺ·еҸ–еҮҪж•°и°ғз”Ёзҡ„е Ҷж Ҳ
  • еўһеҠ дәҶдҝқз•ҷеӯ—пјҲжҜ”еҰӮprotectedгҖҒstaticе’Ңinterfaceпјү

дёҠйқўиҝҷдәӣйҷҗеҲ¶пјҢжЁЎеқ—йғҪеҝ…йЎ»йҒөе®ҲгҖӮз”ұдәҺдёҘж јжЁЎејҸжҳҜ ES5 еј•е…Ҙзҡ„пјҢдёҚеұһдәҺ ES6пјҢжүҖд»ҘиҜ·еҸӮйҳ…зӣёе…і ES5 д№ҰзұҚпјҢжң¬д№ҰдёҚеҶҚиҜҰз»Ҷд»Ӣз»ҚдәҶгҖӮ

е…¶дёӯпјҢе°Өе…¶йңҖиҰҒжіЁж„Ҹthisзҡ„йҷҗеҲ¶гҖӮES6 жЁЎеқ—д№ӢдёӯпјҢйЎ¶еұӮзҡ„thisжҢҮеҗ‘undefinedпјҢеҚідёҚеә”иҜҘеңЁйЎ¶еұӮд»Јз ҒдҪҝз”ЁthisгҖӮ

exportе‘Ҫд»Ө

жЁЎеқ—еҠҹиғҪдё»иҰҒжңүдёӨдёӘе‘Ҫд»Өжһ„жҲҗпјҡexportе’ҢimportгҖӮ

exportз”ЁдәҺи§„е®ҡжЁЎеқ—зҡ„еҜ№еӨ–жҺҘеҸЈпјҢimportе‘Ҫд»Өз”ЁдәҺиҫ“е…Ҙе…¶д»–жЁЎеқ—жҸҗдҫӣзҡ„еҠҹиғҪгҖӮ

дёҖдёӘжЁЎеқ—е°ұжҳҜдёҖдёӘзӢ¬з«Ӣзҡ„ж–Үд»¶гҖӮиҜҘж–Үд»¶еҶ…йғЁзҡ„жүҖжңүеҸҳйҮҸпјҢеӨ–йғЁж— жі•иҺ·еҸ–гҖӮеҰӮжһңдҪ еёҢжңӣеӨ–йғЁиғҪеӨҹиҜ»еҸ–жЁЎеқ—еҶ…йғЁзҡ„жҹҗдёӘеҸҳйҮҸпјҢе°ұеҝ…йЎ»дҪҝз”Ёexportе…ій”®еӯ—иҫ“еҮәиҜҘеҸҳйҮҸгҖӮ

дёҠйқўд»Јз ҒжҳҜprofile.jsж–Үд»¶пјҢдҝқеӯҳдәҶз”ЁжҲ·дҝЎжҒҜгҖӮES6 е°Ҷе…¶и§ҶдёәдёҖдёӘжЁЎеқ—пјҢйҮҢйқўз”Ёexportе‘Ҫд»ӨеҜ№еӨ–йғЁиҫ“еҮәдәҶдёүдёӘеҸҳйҮҸгҖӮ

exportзҡ„еҶҷжі•пјҢйҷӨдәҶеғҸдёҠйқўиҝҷж ·пјҢиҝҳжңүеҸҰеӨ–дёҖз§ҚгҖӮ

жҺЁиҚҗдҪҝз”ЁиҝҷдёҖз§ҚиҜӯжі•гҖӮ

е®ғйҷӨдәҶиҫ“еҮәеҸҳйҮҸпјҢиҝҳеҸҜд»Ҙиҫ“еҮәеҮҪж•°жҲ–зұ»

йҖҡеёёжғ…еҶөдёӢпјҢexportиҫ“еҮәзҡ„еҸҳйҮҸе°ұжҳҜжң¬жқҘзҡ„еҗҚеӯ—пјҢдҪҶжҳҜеҸҜд»ҘдҪҝз”Ёasе…ій”®еӯ—йҮҚе‘ҪеҗҚгҖӮ

дёҠйқўд»Јз ҒдҪҝз”Ёasе…ій”®еӯ—пјҢйҮҚе‘ҪеҗҚдәҶеҮҪж•°v1е’Ңv2зҡ„еҜ№еӨ–жҺҘеҸЈгҖӮйҮҚе‘ҪеҗҚеҗҺпјҢv2еҸҜд»Ҙз”ЁдёҚеҗҢзҡ„еҗҚеӯ—иҫ“еҮәдёӨж¬ЎгҖӮ

жіЁж„Ҹпјҡexportе‘Ҫд»Өи§„е®ҡжҳҜеҜ№еӨ–зҡ„жҺҘеҸЈпјҢеҝ…йЎ»дёҺжЁЎеқ—еҶ…йғЁзҡ„еҸҳйҮҸе»әз«ӢдёҖдёҖеҜ№еә”е…ізі»гҖӮ

жҠҘй”ҷзҡ„еҺҹеӣ е°ұжҳҜ1жҳҜдёҖдёӘеҖјпјҢе®ғдёҚжҳҜдёҖдёӘжҺҘеҸЈгҖӮжӯЈзЎ®зҡ„жҳҜиҝҷж ·

дёҠйқўдёүз§ҚеҶҷжі•йғҪжҳҜжӯЈзЎ®зҡ„пјҢи§„е®ҡдәҶеҜ№еӨ–зҡ„жҺҘеҸЈmгҖӮе…¶д»–и„ҡжң¬еҸҜд»ҘйҖҡиҝҮиҝҷдёӘжҺҘеҸЈпјҢеҸ–еҲ°еҖј1гҖӮе®ғд»¬зҡ„е®һиҙЁжҳҜпјҢеңЁжҺҘеҸЈеҗҚдёҺжЁЎеқ—еҶ…йғЁеҸҳйҮҸд№Ӣй—ҙпјҢе»әз«ӢдәҶдёҖдёҖеҜ№еә”зҡ„е…ізі»гҖӮ

еҗҢж ·зҡ„пјҢfunctionе’Ңclassзҡ„иҫ“еҮәпјҢд№ҹеҝ…йЎ»йҒөе®Ҳиҝҷж ·зҡ„еҶҷжі•гҖӮ

еҸҰеӨ–пјҢexportиҜӯеҸҘиҫ“еҮәзҡ„жҺҘеҸЈпјҢдёҺе…¶еҜ№еә”зҡ„еҖјжҳҜеҠЁжҖҒз»‘е®ҡе…ізі»пјҢеҚійҖҡиҝҮиҜҘжҺҘеҸЈпјҢеҸҜд»ҘеҸ–еҲ°жЁЎеқ—еҶ…йғЁе®һж—¶зҡ„еҖјгҖӮ

дёҠйқўд»Јз Ғиҫ“еҮәеҸҳйҮҸfooпјҢеҖјдёәbarпјҢ500 жҜ«з§’д№ӢеҗҺеҸҳжҲҗbazгҖӮ

з®ҖеҚ•зҗҶи§Је°ұжҳҜпјҡеҜ№иұЎжҡҙйңІпјҢж•ҙдҪ“иөӢеҖјжҡҙйңІпјҢ

е°ҸйҮҚзӮ№е°ұжҳҜпјҢжҺҘеҸЈдёӯзҡ„еҸҳйҮҸеҖјеҠЁжҖҒеҸ‘з”ҹж”№еҸҳпјҢеј•з”ЁиҜҘеҖјзҡ„ж–Үд»¶д№ҹдјҡи·ҹйҡҸеҸҳеҢ–гҖӮ

жңҖеҗҺпјҢexportе‘Ҫд»ӨеҸҜд»ҘеҮәзҺ°еңЁжЁЎеқ—зҡ„д»»дҪ•дҪҚзҪ®пјҢеҸӘиҰҒеӨ„дәҺжЁЎеқ—йЎ¶еұӮе°ұеҸҜд»ҘгҖӮеҰӮжһңеӨ„дәҺеқ—зә§дҪңз”ЁеҹҹеҶ…пјҢе°ұдјҡжҠҘй”ҷпјҢдёӢдёҖиҠӮзҡ„importе‘Ҫд»Өд№ҹжҳҜеҰӮжӯӨгҖӮиҝҷжҳҜеӣ дёәеӨ„дәҺжқЎд»¶д»Јз Ғеқ—д№ӢдёӯпјҢе°ұжІЎжі•еҒҡйқҷжҖҒдјҳеҢ–дәҶпјҢиҝқиғҢдәҶ ES6 жЁЎеқ—зҡ„и®ҫи®ЎеҲқиЎ·гҖӮ

дёҠйқўд»Јз ҒдёӯпјҢexportиҜӯеҸҘж”ҫеңЁеҮҪж•°д№ӢдёӯпјҢз»“жһңжҠҘй”ҷгҖӮ

жіЁйҮҠпјҡиҝҷйҮҢзҡ„йЎ¶еұӮе°ұжҳҜеӨ–еұӮпјҢзҗҶи§ЈдёәжңҖеӨ–еұӮзҡ„ж„ҸжҖқгҖӮ
